# Word to PDF API

A complete .NET 10 REST API for converting Word documents to PDF with template variable replacement and image insertion. Built with open-source libraries and no licensing requirements.

## Features

✅ **Template Variable Replacement** - Replace placeholders like `{{Name}}`, `{{Amount}}`, etc.
✅ **Bookmark Support** - Use Word bookmarks for precise content replacement
✅ **Image Replacement** - Replace images by bookmark reference
✅ **PDF Conversion** - Convert Word documents to PDF using LibreOffice
✅ **REST API** - Easy-to-use HTTP endpoints
✅ **100% Open Source** - No licensing costs or restrictions
✅ **Cross-Platform** - Works on Windows, Linux, and macOS

## Prerequisites

- .NET 10.0 SDK or later
- LibreOffice (free, open-source)
